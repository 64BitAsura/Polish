# Polish Declination Trainer 🇵🇱

An interactive teaching tool for learning Polish noun declination across all 7 grammatical cases. Designed to build muscle memory through repetition and practice.

**[Try it live on GitHub Pages →](https://64bitasura.github.io/Polish/)**

## Features

- **📖 Learn** — Study each of the 7 Polish cases with rules, endings, and example sentences
- **✏️ Practice** — Interactive quizzes with instant feedback to build muscle memory
- **📋 Reference** — Complete declination tables for quick lookup
- **📊 Progress Tracking** — Track your accuracy and streak across sessions

## The 7 Polish Cases

| # | Polish Name | English Name | Question |
|---|------------|-------------|----------|
| 1 | Mianownik | Nominative | kto? co? |
| 2 | Dopełniacz | Genitive | kogo? czego? |
| 3 | Celownik | Dative | komu? czemu? |
| 4 | Biernik | Accusative | kogo? co? |
| 5 | Narzędnik | Instrumental | kim? czym? |
| 6 | Miejscownik | Locative | o kim? o czym? |
| 7 | Wołacz | Vocative | o! |

## How to Use

1. Start with **Learn** to study the rules and examples for each case
2. Move to **Practice** to test yourself with interactive quizzes
3. Use **Reference** as a quick lookup table during study
4. Repeat quizzes to build muscle memory — track your progress on the home page

## Running Locally

Simply open `index.html` in a web browser. No build step or server required.

## Deployment

This project is designed to run on GitHub Pages. Enable GitHub Pages in your repository settings, pointing to the root of the `main` branch.
